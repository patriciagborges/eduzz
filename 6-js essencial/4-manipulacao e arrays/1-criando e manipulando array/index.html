Array.of: Cria uma nova instância de array a partir do numero de parametros informados. Array: Cria uma nova instância de acordo com os parâmetros informados. Array.from: Cria uma nova instância de array de um parâmetro array-like ou iterable object. Inserir e remover elementos: Push: Adiciona um ou mais elementos no final do array e retorna o tamanho do novo array. Pop: Remove o ultimo elemento de um array e retorna o elemento removido. Unshift Adiciona um ou mais elementos no inicio do array. shift Remove o primeiro elemento do array. Concat: Concatena um ou mais arrays, transformando em um array só. Slice: Retorna um novo array "fatiando" o array de acordo com o inicio e fim. Splace: Alterar o array adicionando ou removendo novos elementos.
