Definição Linguagem de programação é uma çlinguagem escrita e formal que especifica um conjunto de instruções e regraas usadas para gerar programas (software). Um software pode ser desenvolvido para rodar em um computador, dispositivo móvel ou em qualquer equipamento que permita sua execução. A função das linguagens de programação é servir de um meio de comunicação entre computadores e humanos. Existem 2 tipos de linguagens de programação alto e baixo nível Alto nível: São as que a sintaxe se aproxima mais da nossa linguagem e de distanciam da linguagem da máquina Baixo nível: É aquela que se aproxima mais da linguagem de máquina. Essas são as que você precisa ter um conhecimento direto da arquitetura do computador para fazer alguma coisa. Linguagens compiladas e interpretadas compiladas É uma linguagem de programação em que o código é executado diretamente pelo sistema operacional ou pelo processador, após ser traduzido por meio de um processo chamado compilação. interpretadas É uma linguagem de programação em que o código fonte é executado por um programa de computador chamado interpretador e em seguida é interpretada pelo sistema operacional ou processador. (js, php, phyton) O que é o portugol? É uma pseudolinguagem que permite ao leitor desenvolver algoritmos estruturados em português de forma simples e intuitiva, independentemente da linguagem de programação.
